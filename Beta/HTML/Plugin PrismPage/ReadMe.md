# Plugin Include
This plugin is part of D2Bridge Framework.
Plugin was designed to help include $prismpage or $prismpage in Template page. With this plugin is possible view preview HTML page also in separe file HTML.

## Installation
- Add include-html.js file in \js folder
- In before final page add script with local file <script src="js/prismpage.js"></script>


## Usage
- Add a div element and put html file name in include-html attribute, Ex: <prismpage page="dashboard.html"></prismpage>
- In this plugin content exist two section, frist not need of JQuery and another need JQuery
